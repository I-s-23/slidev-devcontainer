## usage

- npm modules install to local

  ```bash
    npm i
  ```

```bash
  npm run slidev
```

- check slide with local browser 

  ```bash
    npm run slidev test.md
  ```

- export pdf to slide file 

  ```bash
    npm run slidev export test.md
  ```
